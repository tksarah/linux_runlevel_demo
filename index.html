initコマンドでランレベルを変更する

ターミナルで init 0、init 1、init 3、 init 5、init 6 を実行し、Linuxの状態変化を確認します。

STEP 1

initコマンドを入力

ボタンで選ぶか、端末欄に直接入力してください。

#

入力例: init 5

STEP 3

現在の状態

Selected command / init 0

Runlevel 0

システムを停止します。ログイン画面には到達しません。

STEP 2

起動ログ

ターミナルで init 数字 を実行してください。

例:
# init 0

STEP 4

見た目の変化

REFERENCE

簡単な解説

Runlevel 0: halt / システム停止

init 0は停止処理です。サービスを止め、ファイルシステムを安全に処理し、 最後に電源断できる状態へ向かいます。

補足: 現在のLinuxではsystemd targetで同じような起動状態を扱う環境も多くあります。
